SQL SELECT - 查詢資料表資料

語法說明

SELECT 用於從資料表中查詢資料。其基本語法如下：

SELECT column_name FROM table_name;

範例參數：

  • column_name：要查詢的欄位
  • table_name：目標資料表

實務案例

假設我們要查詢 employees 表中的所有員工姓名。語法如下：

SELECT name FROM employees;

假設表已有資料，查詢結果如下：
