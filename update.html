SQL UPDATE - 更新資料表資料

語法說明

UPDATE 用於修改資料表中的現有資料。其基本語法如下：

UPDATE table_name
SET column = value
WHERE condition;

範例參數：

  • table_name：目標資料表
  • column：要更新的欄位
  • condition：過濾條件

實務案例

假設我們要將 employees 表中員工張三的薪資改為60000。語法如下：

UPDATE employees
SET salary = 60000.00
WHERE name = '張三';

執行後，資料表內容更新如下：

id name hire_date salary
1 張三 2023-01-01 60000.00

張三的薪資已成功更新。
