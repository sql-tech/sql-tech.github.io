SQL ALTER - 修改資料表結構

語法說明

ALTER 用於修改現有的資料表結構，例如新增欄位、修改欄位型態或約束。其基本語法如下：

ALTER TABLE table_name
ADD column_name datatype constraints;

範例參數：

  • table_name：資料表名稱
  • column_name：新欄位名稱
  • datatype：資料型態（如DECIMAL、VARCHAR）

實務案例

假設我們要為 employees 表新增一個 salary 欄位，儲存員工薪資。語法如下：

ALTER TABLE employees
ADD salary DECIMAL(10,2);

執行後，資料表結構更新如下：

欄位名稱 資料型態 約束條件
id INT PRIMARY KEY
name VARCHAR(50) NOT NULL
hire_date DATE -
salary DECIMAL(10,2) -

新增的 salary 欄位可用於記錄員工薪資，支援兩位小數。
