SQL SET - 設定環境變數

語法說明

SET 用於設定資料庫的環境變數或參數（視系統而定）。其基本語法如下：

SET variable = value;

範例參數：

  • variable：要設定的變數
  • value：設定值

實務案例

假設我們要設定交易隔離等級為 READ COMMITTED。語法如下（以SQL Server為例）：

SET TRANSACTION ISOLATION LEVEL READ COMMITTED;

執行後，當前會話的交易隔離等級被設定，影響資料一致性行為。
