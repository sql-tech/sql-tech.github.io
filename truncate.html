SQL TRUNCATE - 清空資料表

語法說明

TRUNCATE 用於清空資料表中的所有資料，但保留表結構。其基本語法如下：

TRUNCATE TABLE table_name;

範例參數：

  • table_name：要清空的資料表名稱

實務案例

假設我們要清空 employees 表中的所有資料。語法如下：

TRUNCATE TABLE employees;

執行後，employees 表將變為空表，但結構保留如下：

欄位名稱 資料型態 約束條件
id INT PRIMARY KEY
name VARCHAR(50) NOT NULL
hire_date DATE -

所有資料被移除，但表仍可繼續使用。
