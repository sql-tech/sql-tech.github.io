SQL DROP - 刪除資料表或資料庫

語法說明

DROP 用於永久刪除資料表或資料庫，執行後無法復原。其基本語法如下：

DROP TABLE table_name;

範例參數：

  • table_name：要刪除的資料表名稱

實務案例

假設我們要刪除 employees 資料表。語法如下：

DROP TABLE employees;

執行後，employees 表將被完全移除，原本的結構與資料不再存在。

注意：執行前請確認是否需要備份資料。
