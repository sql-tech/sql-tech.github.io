SQL DELETE - 刪除資料表資料

語法說明

DELETE 用於從資料表中刪除特定資料。其基本語法如下：

DELETE FROM table_name
WHERE condition;

範例參數：

  • table_name：目標資料表
  • condition：過濾條件

實務案例

假設我們要刪除 employees 表中員工張三的資料。語法如下：

DELETE FROM employees
WHERE name = '張三';

執行後，employees 表將變為空表（假設只有這一筆資料）。
