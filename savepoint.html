SQL SAVEPOINT - 設定交易保存點

語法說明

SAVEPOINT 用於在交易中設定一個保存點，以便部分回滾。其基本語法如下：

SAVEPOINT savepoint_name;

範例參數：

  • savepoint_name：保存點名稱

實務案例

假設我們在交易中新增資料並設定保存點。語法如下：

INSERT INTO employees (id, name, hire_date) VALUES (4, '趙六', '2023-04-01');
SAVEPOINT sp1;
DELETE FROM employees WHERE id = 4;
ROLLBACK TO SAVEPOINT sp1;

執行後，刪除操作被取消，趙六 的資料保留。
