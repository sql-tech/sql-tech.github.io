SQL INSERT - 新增資料到資料表

語法說明

INSERT 用於向資料表中新增資料。其基本語法如下：

INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);

範例參數：

  • table_name：目標資料表
  • column1, column2：欄位名稱
  • value1, value2：對應值

實務案例

假設我們要向 employees 表新增一筆員工資料。語法如下：

INSERT INTO employees (id, name, hire_date, salary)
VALUES (1, '張三', '2023-01-01', 50000.00);

執行後，資料表內容如下：

id name hire_date salary
1 張三 2023-01-01 50000.00

這筆資料代表員工張三的基本資訊已成功新增。
